# GiftCraft

GiftCraft is a standalone React app ready for Vercel.

## Local Setup

```bash
npm run build
```

## Vercel Settings

- Framework Preset: Other
- Build Command: `npm run build`
- Output Directory: `dist`

The build copies the standalone `index.html` into `dist/index.html`, so no extra npm packages are required.

The app uses Google Fonts and Unsplash images, so the deployed site needs browser internet access for the full visual design.
